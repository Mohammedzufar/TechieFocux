# Techie Focux

Focus and task manager with Pomodoro timer, ambient music (piano & nature), and light/dark mode.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

Output is in the `dist` folder.

## Deploy on Netlify

Connect this repo on [Netlify](https://app.netlify.com). Settings are in `netlify.toml`:

- **Build command:** `npm run build`
- **Publish directory:** `dist`
